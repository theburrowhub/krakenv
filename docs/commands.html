Getting Started Syntax Commands GitHub

Commands Reference

Global Flags

--dist, -d Path to distributable file .env.dist
--non-interactive, -n Disable TUI; fail on unresolved false
--quiet, -q Suppress non-error output false
--verbose, -v Enable detailed output false

generate

Generate or update an environment file from the distributable.

krakenv generate <target> [flags]

Flags

--force, -f Overwrite without confirmation
--all, -a Generate all environments from config
--keep-annotations, -k Preserve annotations in output

Examples

# Interactive mode
krakenv generate .env.local

# Generate all environments
krakenv generate --all

# CI/CD mode (fails if unresolved)
krakenv generate .env.local --non-interactive

validate

Validate environment file against distributable annotations.

krakenv validate <target> [flags]

Flags

--strict, -s Require all variables to have annotations

Exit Codes

0All validations passed
1Validation errors found
2File not found

Examples

krakenv validate .env.local
krakenv validate .env.production --strict
krakenv validate .env.testing --non-interactive

inspect

Compare environment file with distributable; identify discrepancies.

krakenv inspect <target> [flags]

Flags

--sync, -s Interactively sync discrepancies
--json, -j Output as JSON

Examples

krakenv inspect .env.local
krakenv inspect .env.local --json | jq '.missing | length'

add

Add a new annotated variable to the distributable.

krakenv add <name> [flags]

Flags

--type, -tVariable type
--prompt, -pPrompt message
--default, -DDefault value
--minMinimum (int/numeric)
--maxMaximum (int/numeric)
--minlenMin length (string)
--maxlenMax length (string)
--patternRegex pattern (string)
--options, -oOptions (enum)
--formatFormat (object)
--optionalMark as optional
--secretMark as secret

Examples

krakenv add API_URL --type string --prompt "API base URL?"
krakenv add MAX_CONN --type int --min 1 --max 100 --default 10
krakenv add LOG_LEVEL --type enum --options "debug,info,warn,error"
krakenv add DB_PASSWORD --type string --secret

init

Initialize a new distributable file.

krakenv init [flags]

Flags

--path, -pOutput path (default: .env.dist)
--environments, -eEnvironments (default: local)
--template, -tCreate with example comments only
--force, -fOverwrite existing file

version

Display version and build information.

krakenv version

- name: Validate environment
