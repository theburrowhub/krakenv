Getting Started

Installation

Homebrew (macOS/Linux)

brew install theburrowhub/tap/krakenv

Go Install

go install github.com/theburrowhub/krakenv/cmd/krakenv@latest

Binary Download

Download pre-built binaries from the Releases page.

Quick Start

1. Initialize a Distributable

krakenv init

This creates a .env.dist file with example structure.

2. Add Variables

# Add a string variable
krakenv add API_URL --type string --prompt "API base URL?"

# Add an integer with constraints
krakenv add MAX_CONNECTIONS --type int --min 1 --max 100 --default 10

# Add an enum
krakenv add LOG_LEVEL --type enum --options "debug,info,warn,error" --default info

# Add a secret
krakenv add DB_PASSWORD --type string --prompt "Database password?" --secret

3. Generate Environment File

krakenv generate .env.local

The wizard will prompt for each variable that needs a value.

4. Validate Configuration

krakenv validate .env.local

Example Workflow

Your .env.dist file:

#krakenv:environments=local,testing,production

# Database
DB_HOST=localhost #prompt:Database host?|string
DB_PORT=5432 #prompt:Database port?|int;min:1;max:65535
DB_PASSWORD= #prompt:Password?|string;secret

# App
APP_ENV=development #prompt:Environment?|enum;options:development,staging,production

Multi-Environment Setup

Generate all environments at once:

krakenv generate --all

This reads the environments config from your distributable and generates:

  • .env.local
  • .env.testing
  • .env.production
